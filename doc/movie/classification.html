Description

The classification element describes the film, by identifying whether it's fiction or non-fiction, the genres or type of film it is, any specific subjects the film covers, and any ratings certifications the film may have.

Genres Elements

Element<genres>
Module vtm-mod-visual-classification.xsd
UsageOptional, one occurence max
Parent Element <classification>
Child Elements <primary> <secondary> <setting> <specific> <subgenres>

The genres element contain all elements used in describing the genre of the movie. It is designed to be as flexible (or as rigid) as the implementor needs. The only child element that is required is the primary element; all others are optional. Although the usage of all the tags can help create a more rounded categorization of the type of movie.

Subjects Element

Element<subjects>
Module vtm-mod-visual-classification.xsd
UsageOptional, one occurence max
Parent Element <classification>
Child Elements <generic> <properNoun>

Subjects are primarily used for non-fictional films, describing one or more subjects that are the primary focus of the film. For an example, an autobiography of Florence Bascom, the subject is obviously Florence Bascom. Generic terms like botany are filed using the <generic> element, where proper nouns use the <properNoun> element.

The schema does not enforce a limit, but the recommendation is that a film has no more than four subjects.

Element<generic>
Module vtm-mod-visual-classification.xsd
UsageOptional, one or more occurences
Valuesubject phrase (xs:token)
Parent Element <subjects>
Child Elements None

A generic subject term, like skydiving, bass fishing, or hospital ward. They are typically nouns.

Container element for a proper noun. See the proper noun elements for more details.

Subject Element: Proper Nouns

Proper nouns are specific references to individual nouns. In the context of a classification subject, proper nouns have a higher distinction over generic nouns because they are more specific, and it's easier for an end user to search for the exact material they are looking for.

Certification Elements

Element<certifications>
Module vtm-mod-visual-classification.xsd
UsageOptional, one occurence max
Parent Element <classification>
Child Elements <certificate>

References all ratings certificates that have been issued for the film in question.

Element<certificate>
Module vtm-mod-visual-classification.xsd
UsageRequired, one or more occurences
Parent Element <certifications>
Child Elements<rating> <agency>
AttributesNameTypeUsage
refstring (xs:token)optional

A certificate references a specific certificate issued by a rating agency. The element references the issued rating, and the information on the agency that issued it. The ref attribute references a unique identifier that some agencies use to reference the certificate. For example, the Motion Picture Association has a 5 digit number issued to all films they rate.
